# 项目介绍

本项目是一个仿社交平台，主要功能浏览、发布帖子。

## 技术栈

+ 前端：Next.js + Zustand + next-auth + next-intl + Tailwind CSS
+ 后端: Nest.js + Prisma + MySql 
+ Api: GraphQL + Apollo Client + Apollo Server

## 实现功能

+ 完成贴子、点赞、评论、转发、收蔽等模块(40+接口)
+ 封装 Message、 Tooltip、Skeleton 等组件
+ 语言国际化、主题多样化
